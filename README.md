## üöÄ Getting Started

### Prerequisites

*   Node.js 18+
*   A Google Gemini API Token

### Installation

1.  **Clone the Cluster**
    ```bash
    git clone https://github.com/ROHITH-NATHANI/ResuMaster-Pro.git
    cd ResuMaster-Pro
    ```

2.  **Ignite Dependencies**
    ```bash
    npm install
    ```

3.  **Secure Configuration**
    Create a `.env` file in the root directory:
    ```env
    GEMINI_API_TOKEN=your_actual_api_token_here
    ```
    > ‚ö†Ô∏è **Security Note**: Never commit your `.env` file. It is already added to `.gitignore`.

4.  **Launch Sequence**
    ```bash
    npm run dev
    ```
    Access the interface at `http://localhost:3000`.
